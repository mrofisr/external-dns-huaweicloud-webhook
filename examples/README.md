# Examples

This directory contains example Kubernetes manifests demonstrating how to deploy and use the ExternalDNS HuaweiCloud webhook.

| Example | Description |
|---------|-------------|
| [basic-deployment](basic-deployment/) | Deploy ExternalDNS with the HuaweiCloud webhook using IDP token auth |
| [static-credentials](static-credentials/) | Deploy using static AK/SK credentials |
| [service-dns](service-dns/) | Create DNS records from a LoadBalancer Service |
| [ingress-dns](ingress-dns/) | Create DNS records from an Ingress resource |
